# PlanetPangaea.jl
Climate modelling for dinosaurs. 

## Contents

Script for generating Pangaea-like boundary-conditions NetCDF files from a sample input image for use in `SpeedyWeather.jl` simulations.

Minimum boundary-conditions files required:
- Albedo
- Land-sea mask
- Orography
- Soil moisture *(seasonal)*
- Sea-surface temperature *(seasonal)*
- Vegetation
